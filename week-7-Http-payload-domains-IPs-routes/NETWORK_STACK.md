## Network Protocol stack

OSI ( 7 layers) 
1. Application: End applications like: chat appn, email service
2. Presentation: Presentation of Data, compression, encryption
3. Session: User session management (user login ets)
4. Transport: divides big chunk of data into small chunks
5. Network: Handles routing of packets 
6. Data Link: error/flow control, multiplexing and demultiplexing, handle address
7. Physical: 

TCP/IP (5 Layers)
